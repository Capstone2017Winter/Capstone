1.) Install pip "sudo apt-get install python3-pip"
2.) Install Django "pip3 install Django"
3.) Install gunicorn "pip3 install gunicorn"
4.) Install nginx "sudo apt-get install nginx"
5.) Configure nginx, create the file /etc/nginx/sites-enabled/schedu and then paste this into it:
server {
        listen localhost:8001;

        location / {
                proxy_pass http://127.0.0.1:8000;
        }

        location /static/ {
                autoindex on;
                alias /home/elias/Capstone/src/schedu/schedule_builder/static/;
        }
}
